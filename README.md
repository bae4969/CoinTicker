 - 설명
    트레이 아이콘과 반투명 창으로 실시간 업비트 코인 데이터를 확인할 수 있는 프로그램입니다.

 - 기능

1. Chart
    - 해당 코인에 대한 차트가 나오게 됩니다.
    - UpbitDealer에서 가져온 차트입니다.

2. Add Ticker
    - 계속 시세를 확인할 코인을 추가합니다.

3. Remove Ticker
    - Add Ticker로 추가한 코인을 제거합니다.

4. Opacity
    - Ticker창의 불투명도를 설정합니다.

5. Exit
    - 프로그램을 종료합니다.


 - 기본 조작
    -아이콘을 왼쪽 클릭하면 메뉴가 나오게 되고 오른쪽 클릭하면 메뉴가 사라집니다.
